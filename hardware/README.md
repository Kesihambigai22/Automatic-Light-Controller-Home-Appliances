# Hardware Implementation

This folder contains the hardware design, circuit implementation, and prototype images for the Automatic Light Controller project.

---

## Circuit Diagram

`circuit.png`

The circuit consists of:
- IR sensor pair
- LM358 comparator
- CD4017 decade counter
- BC547 transistor
- Relay driver circuit
- LED indicators

## Hardware Features

- Automatic light switching
- Motion detection
- Relay-based AC load control
- Low power consumption
- No microcontroller required

---

## Power Requirements

- Input Supply: 5V DC
- Relay Voltage: 5V
- AC Load Support: 110V/220V AC bulb
